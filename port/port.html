Rock, Paper, Scissors

This project is a simulation of the classic game of rock, paper, scissors. It showcases my ability to use variables, conditionals, and functions to create an interactive program. The game randomly selects rock, paper, or scissors for each player and determines the winner based on the traditional rules.

Cash Register Project

This project involves designing and building a cash register simulation. The program is capable of adding and removing items, counting the amount of coins and bills in the drawer, summing the total amount of money in the drawer, determining if a specific cash amount can be created from the current items in the drawer, and computing the change required from a transaction and removing it from the drawer if possible.
